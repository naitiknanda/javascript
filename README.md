# JavaScript Logic, Math Modules & Chaos

Welcome to my personal JavaScript testing ground. This repository is where I break things, fix them, and experiment with the core logic of the web. From simple arithmetic engines to "E-Waste" optimization scripts, if it’s written in JS, it’s here.

    Current Status: 🛠️ Active Development / Experimental

🚀 What’s Inside?

    Math Engines: Simple calculators and logic modules to handle "the boring stuff."

    Logic Tests: Scripts designed to understand how the JS engine handles variables and loops.

    The "Dump" Files: Raw, unedited code snippets where I test new ideas before they go into Softsense or The HQ.

    Vanilla JS: No heavy frameworks—just pure, raw JavaScript.

📉 Why This Repo Exists

Every great developer has a place where they aren't afraid to fail. This is mine.

    Goal 1: Master the DOM and Asynchronous logic.

    Goal 2: Translate complex math into functional code.

    Goal 3: Keep the "JavaScript EEEEEEEEEEE" energy alive.

🛠️ Technical Specs

    Language: JavaScript (ES6+)

    Environment: Browser / Node.js

    Level: From "Hello World" to "How did I fix that?"
